# Smart Todo App

A simple and responsive Smart Todo Web Application built using HTML, CSS, and Vanilla JavaScript.

This project helps users manage daily tasks with features like adding, editing, deleting, searching, filtering, and saving tasks using Local Storage.

---

# Live Demo

🔗 Live Website: https://priyanka-smart-todo.netlify.app/

# Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Task priority levels (High, Medium, Low)
- Search tasks
- Filter tasks
  - All Tasks
  - Completed Tasks
  - Pending Tasks
- Task counter
- Data saved using Local Storage
- Responsive design

# How It Works

## Add Task
Users can enter a task and select a priority level.

## Complete Task
Tasks can be marked as completed using a checkbox.

## Edit Task
Users can update existing tasks.

## Delete Task
Tasks can be removed permanently.

## Local Storage
All tasks are stored in the browser using localStorage, so tasks remain even after refreshing the page.
